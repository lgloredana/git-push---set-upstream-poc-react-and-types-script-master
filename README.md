# React Redux TypeScript

This project allows to :
 - manage TODO's: 
   * add
   * remove
- manage GOAL's: 
    * add
    * remove   
    
 This is done using react-redux-typeScript technologies
 
 
 ## For starting the project run : 
  * **npm install**  
  * **npm start**
  
  or
  
  * **yarn install**  
  * **yarn start**
